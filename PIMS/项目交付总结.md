## 🚀 快速启动

### 1. 初始化数据库
```cmd
# 使用CMD执行
cd d:\mypro\PIMS
mysql -uroot -p123456 < pims.sql
```

### 2. 启动后端
```cmd
# 方式1：IDE运行
打开 PimsApplication.java -> 右键Run

# 方式2：Maven运行
cd pims-backend
mvn clean package
java -jar target/pims-backend-1.0.0.jar
```

### 3. 访问接口文档
http://localhost:8080/api/doc.html

### 4. 测试核心功能
参考《快速启动和测试指南.md》第3节
