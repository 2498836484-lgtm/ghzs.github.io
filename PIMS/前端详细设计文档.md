# 物业管理系统（PIMS）前端详细设计文档

## 1. 系统概述

### 1.1 技术选型
- **前端框架**: Vue 3.3+
- **构建工具**: Vite 4.x
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router 4.x
- **HTTP客户端**: Axios
- **工具库**: day.js（日期处理）、lodash-es（工具函数）
- **样式**: SCSS
- **图标**: Element Plus Icons

## 2. 页面设计

### 2.1 用户端页面

#### 2.1.1 登录页面 (Login.vue)
**页面结构**
- 顶部：系统Logo和标题
- 中间：登录表单
  - Tab切换：房号登录 / 手机号登录
  - 输入框：房号（或手机号）、密码
  - 记住密码复选框
  - 登录按钮
- 底部：
  - "去注册"链接
  - "找回密码"链接

**验证规则**
- 房号格式：Xxxx（如A310）
- 手机号格式：11位中国大陆手机号
- 密码：非空验证

**交互逻辑**
- 登录成功后跳转到入住申请页面或用户主页（根据入住状态）
- 登录失败显示错误提示

#### 2.1.2 注册页面 (Register.vue)
**页面结构**
- 标题："用户注册"
- 表单字段：
  - 手机号输入框（必填）
  - 房号输入框（必填）
  - 密码输入框（必填）
  - 确认密码输入框（必填）
  - 验证码输入框（必填）
  - 获取验证码按钮（60秒倒计时）
- 提交按钮
- "返回登录"链接

**验证规则**
- 手机号：11位，符合中国大陆手机号格式
- 房号：Xxxx格式，如A310
- 密码：6-20位，包含字母和数字
- 确认密码：与密码一致
- 验证码：6位数字

**交互逻辑**
- 获取验证码：验证手机号格式后启用，60秒倒计时
- 提交注册：所有验证通过后提交，成功后跳转登录页
- 房号重复提示："该房号已被注册"

### 3.2 路由守卫
```javascript
router.beforeEach((to, from, next) => {
  const token = getToken()
  const userType = getUserType() // 'user' or 'admin'
  
  // 需要认证的路由
  if (to.meta.requiresAuth) {
    if (!token) {
      // 未登录，跳转到对应登录页
      if (to.meta.requiresAdmin) {
        next('/admin/login')
      } else {
        next('/user/login')
      }
      return
    }
  }
  
  // 需要管理员权限
  if (to.meta.requiresAdmin && userType !== 'admin') {
    next('/admin/login')
    return
  }
  
  // 需要已入住用户权限
  if (to.meta.requiresResident) {
    const residentStatus = getResidentStatus()
    if (residentStatus !== 1) {
      next('/user/apply')
      return
    }
  }
  
  next()
})
```

## 4. 状态管理（Pinia）

### 4.1 用户状态 (user.js)
```javascript
export const useUserStore = defineStore('user', {
  state: () => ({
    token: localStorage.getItem('user_token') || '',
    userInfo: JSON.parse(localStorage.getItem('user_info') || '{}'),
    residentInfo: JSON.parse(localStorage.getItem('resident_info') || '{}')
  }),
  
  getters: {
    isLogin: (state) => !!state.token,
    isResident: (state) => state.residentInfo.status === 1
  },
  
  actions: {
    setToken(token) {
      this.token = token
      localStorage.setItem('user_token', token)
    },
    
    setUserInfo(info) {
      this.userInfo = info
      localStorage.setItem('user_info', JSON.stringify(info))
    },
    
    setResidentInfo(info) {
      this.residentInfo = info
      localStorage.setItem('resident_info', JSON.stringify(info))
    },
    
    logout() {
      this.token = ''
      this.userInfo = {}
      this.residentInfo = {}
      localStorage.removeItem('user_token')
      localStorage.removeItem('user_info')
      localStorage.removeItem('resident_info')
    }
  }
})
```

## 5. API封装

### 5.1 Axios配置 (request.js)
```javascript
const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 10000
})

// 请求拦截器
request.interceptors.request.use(
  config => {
    const userStore = useUserStore()
    const adminStore = useAdminStore()
    
    // 根据请求路径判断使用哪个token
    if (config.url.includes('/admin/')) {
      config.headers.Authorization = `Bearer ${adminStore.token}`
    } else {
      config.headers.Authorization = `Bearer ${userStore.token}`
    }
    
    return config
  },
  error => {
    return Promise.reject(error)
  }
)

// 响应拦截器
request.interceptors.response.use(
  response => {
    const res = response.data
    
    if (res.code !== 200) {
      ElMessage.error(res.message || '请求失败')
      
      // Token过期，跳转登录
      if (res.code === 401) {
        // 清除token，跳转登录页
        if (response.config.url.includes('/admin/')) {
          useAdminStore().logout()
          router.push('/admin/login')
        } else {
          useUserStore().logout()
          router.push('/user/login')
        }
      }
      
      return Promise.reject(new Error(res.message || '请求失败'))
    }
    
    return res
  },
  error => {
    ElMessage.error(error.message || '网络错误')
    return Promise.reject(error)
  }
)
```

```javascript
export const passwordValidator = (rule, value, callback) => {
  if (!value) {
    callback(new Error('请输入密码'))
  } else if (value.length < 6 || value.length > 20) {
    callback(new Error('密码长度为6-20位'))
  } else if (!/^(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{6,20}$/.test(value)) {
```
