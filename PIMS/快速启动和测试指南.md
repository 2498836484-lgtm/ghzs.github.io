### 测试场景1：用户注册和登录流程

#### 1.1 发送验证码
**接口**：`POST /api/user/send-code?phone=13800138000`

**预期结果**：
- 返回成功
- 控制台打印验证码（6位数字）

#### 1.2 用户注册
**接口**：`POST /api/user/register`

**请求体**：
```json
{
  "phone": "13800138000",
  "password": "abc123",
  "roomNumber": "A310",
  "verifyCode": "从控制台复制验证码"
}
```

**预期结果**：
- 返回"注册成功"
- 数据库t_user表新增一条记录

#### 1.3 用户登录（房号登录）

**请求体**：
```json
{
  "account": "A310",
  "password": "abc123"
}
```

**预期结果**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
```

## ⚠️ 常见问题

### 1. 启动失败：数据库连接错误
**解决方法**：
- 检查MySQL是否启动
- 确认数据库pims已创建
- 验证用户名密码（root/123456）

### 2. 依赖下载失败
**解决方法**：
```xml
<!-- 在pom.xml中配置阿里云镜像 -->
<repositories>
    <repository>
        <id>aliyun</id>
        <url>https://maven.aliyun.com/repository/public</url>
    </repository>
</repositories>
```

### 3. 端口8080被占用
**解决方法**：
修改`application.yml`中的端口：
```yaml
server:
  port: 8081
```

### 4. 验证码验证失败
**原因**：验证码使用内存存储，重启后失效
**解决方法**：重新发送验证码
