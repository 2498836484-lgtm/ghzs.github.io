## 📝 测试指南

### 快速测试流程（10分钟）

#### 1. 访问API文档
启动后访问：http://localhost:8080/api/doc.html

#### 2. 测试用户注册流程
**步骤1**：发送验证码
- 接口：`POST /api/user/send-code?phone=13800138000`
- 查看控制台获取验证码

**步骤2**：注册用户
- 接口：`POST /api/user/register`
- 请求体：
```json
{
  "phone": "13800138000",
  "password": "abc123",
  "roomNumber": "A310",
  "verifyCode": "从控制台复制"
}
```

**步骤3**：用户登录
- 接口：`POST /api/user/login/room`
- 请求体：
```json
{
  "account": "A310",
  "password": "abc123"
}
```
- 保存返回的Token

#### 3. 测试入住申请流程
**步骤1**：提交入住申请
- 接口：`POST /api/user/apply-resident?userId=1000`
- 请求体：
```json
{
  "name": "张三",
  "idCard": "110101199001011234",
  "area": 80.5
}
```

**步骤2**：管理员登录
- 接口：`POST /api/admin/login`
- 请求体：
```json
{
  "account": "admin",
  "password": "admin123"
}
```

**步骤3**：查看待审核列表
- 接口：`GET /api/admin/audit/pending-list`

**步骤4**：审核通过
- 接口：`POST /api/admin/audit/process?adminId=1`
- 请求体：
```json
{
  "auditId": 1,
  "status": 1,
  "pricePerSqm": 2.0
}
```

**步骤5**：验证结果
- 接口：`GET /api/user/resident-info?userId=1000`
- 应该能查到住户信息
